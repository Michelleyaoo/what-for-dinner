# Recipe Images

Add your recipe images to this folder:

- `tomato-egg-stir-fry.png` - Image for "Tomato and egg stir fry" recipe (variant 1)
- `cozy-beef-stew.png` - Primary image for "Cozy beef stew" recipe (variant 2)
- `beef-stew-overlay.png` - Overlay image for "Cozy beef stew" recipe (variant 2)
- `thai-red-curry.png` - Image for "Thai red curry with beef" recipe (variant 3)

## Image Specifications

- Recommended size: 240px × 200px (or maintain aspect ratio)
- Format: PNG or JPG
- The images will be automatically optimized by Vite during build

Once you add the images, update the imports in `src/components/Image.jsx`.
